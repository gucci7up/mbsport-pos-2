OBJETIVO

Construir el frontend completo de un POS profesional para apuestas de carreras de galgos llamado MBRACES POS.

IMPORTANTE:

NO rediseñar.
NO modernizar.
NO convertir en dashboard SaaS.
NO usar tarjetas estilo CRM.
NO usar diseño tipo administración.
NO usar layouts responsive móviles.

Debe parecer una terminal real de apuestas utilizada por operadores en una banca física.

Tomar como referencia visual el POS clásico de carreras de galgos.
